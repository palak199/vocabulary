# JetBrains Mono

JetBrains Mono is a monospaced font developed by JetBrains and designed to make reading and writing code easier.

It is used in Vocabulary in 2 font weight: regular (400) and bold (700).

export const sampleCode = "const isEqual = (a, b) => { a === b }"
